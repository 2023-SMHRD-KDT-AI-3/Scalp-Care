## 🤾‍♂️ 트러블 슈팅
* ### 문제1<br>
- <strong>문제점</strong> : 모델링 학습 정확도 결과
- <strong>해결 방안</strong> : <모델 선정과정>

 처음에 사용한 모델은 AIHub데이터 구축 업체가 사용한 efficientnet b7입니다.
efficientnet 모델은 비교적 최근인 2019년에 등장한 모델로 모델의 깊이, 너비, 이미지크기를 능동적으로 조절해주어 적은 파라미터수로 높은 정확도를 보여주는 모델입니다.

 efficientnet 모델로 학습을 진행해보니 저희가 가지고 있는 장비로는 한번 학습에 시간이 매우 많이 소모된다는 문제가 발생했습니다. 약 2만장의 데이터를 1번 학습시키는데 1시간 가까이 소모되는 문제가 발생했습니다. 그렇게 30여번을 학습시켜본 결과 정확도는 약 73%를 보여주었지만 긴 학습시간으로 6개의 모델을 학습시켜야 하는 저희에게는 적합하지 않다는 결론을 내렸습니다.

 그렇게 사용한 2번째 모델은 학습시간을 최대한 줄이기 위해 가장 단순한 CNN모델인 VGG-16입니다. 1회 학습에 약 2분 내외로 적은 시간에 많은 반복학습을 진행 할 수 있었습니다. 하지만 학습을 진행하다보니 각 레이블간의 데이터의 수의 차이가 매우 커서 점차 과대적합현상이 발생하였으며 정확도는 약 70%를 보여주었습니다. 과대적합을 제어하기 위하여 부족한 데이터의 증강 및 각 레이어층에 배치정규화를 진행하였고 학습한 결과 약 300번의 학습 끝에 정확도는 약 75%라는 결과를 얻을 수 있었습니다.

 3번째 모델은 조금더 정확도를 올려보기 위해 VGG-16과 비슷한 구조인 VGG-19를 사용하였습니다. VGG-16과 같이 데이터 증강과 배치정규화를 통해서 약 300번의 학습끝에 정확도는 소폭 상승한 약 77%라는 결과를 얻을 수 있었습니다. 그렇게 VGG-19 모델을 프로젝트에 사용하기로 결정하였습니다.
